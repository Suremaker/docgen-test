    Calculate(IEnumerable<ExecutionTime>)

    Calculates ExecutionTimeSummary for all instances provided in times argument. The Start is set to the lowest value from provided times. The End is set to the highest value from provided times. The Aggregated is set to the sum of all values from provided times. The Average is set to the Aggregated divided by number of values from provided times.

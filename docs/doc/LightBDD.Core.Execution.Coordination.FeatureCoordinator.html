    Dispose()

    Disposes coordinator, triggering feature result aggregation. Each runner belonging to RunnerRepository, is disposed and its feature result is aggregated. After aggregation of all results, the feature aggregator is disposed as well.

    If coordinator is already disposed, methods does nothing.
